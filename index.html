Portfolio Website

This project was built with HTML, CSS, and JavaScript as a fun idea to share with my classmates and to showcase my skills in music production. I created a fully dynamic system without any page refreshes, using JavaScript to switch content with display: none/block and custom logic. The site uses a complex data structure where every bundle contains multiple packs, and each pack contains several instruments, each with its own price. This project helped me learn how to organize data, build interactive UIs, and create a smooth user experience using only frontend technologies.

Music Store

In this project, I upgraded my original music website by connecting it to a complete PHP backend. I implemented a login and logout system, user accounts, and a secure database using PDO. Each user has a balance, and instruments, packs, and bundles can be purchased through a custom payment logic that updates the account in real time. This project taught me how to connect frontend and backend, manage sessions, handle data securely, and build a fully functional system where everything works together.
